# Frontend

Next.js application for the Image2Surface project. Handles image upload, 3D mesh visualization, and editing controls.

## Tech Stack

- Next.js 16.2.4
- React 19
- TypeScript
- Three.js (3D rendering)
- Tailwind CSS
- Radix UI

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm run start
```

## Features

- Image upload (PNG, JPG up to 5MB)
- 3D surface generation from images
- Mesh editing (height scale, smooth, sharpen)
- Interactive 3D viewer with rotate/zoom controls
